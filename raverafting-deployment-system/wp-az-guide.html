02. PARTS & CONNECTIONS

WordPress is a modular machine. Every part connects to others through the **Hooks System**.

  • The Core (`wp-includes`): The engine block. Never modify these files.
  • The Theme (`wp-content/themes`): The skin. It connects to the Core via Template Tags.
  • The Plugins (`wp-content/plugins`): The specialized weaponry. They "hook" into the Core and Theme to intercept data.
  • The Database (MySQL): The memory. It stores everything from Avicii's monthly listeners to your latest casino guides.
  • The `.htaccess`: The perimeter fence. It handles routing and security at the server level.

04. VIRAL RECOVERY PROTOCOL

When the "Virus" hits (Malware, DB corruption, or bad code), execute the **State Reload**.

  1. ISOLATE: Put the site in Maintenance Mode immediately.
  2. RELOAD STATE: Restore the latest "Clean Intel" (XML Export + Database SQL) from a trusted backup.
  3. PURGE CORE: Delete everything in the root directory *except* `wp-config.php` and `wp-content`. Upload a fresh copy of WordPress Core.
  4. HARDEN PERIMETER: Update all passwords, reset Salts in `wp-config.php`, and verify all plugin signatures.
