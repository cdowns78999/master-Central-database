### 🧠 THE 12-PHASE LOGIC
Remember the tactical flow during your deployment:
`URL` -> `Core` -> `Content` -> `Design` -> `Plugins` -> `Browser`.

If a page looks wrong, check the layers in this order.
1. **Broken Link?** Check Core (Layer 1).
2. **Missing Text?** Check Content (Layer 2).
3. **Wrong Color?** Check Design (Layer 3).
4. **Missing Sidebar Tool?** Check Plugins (Layer 4).
