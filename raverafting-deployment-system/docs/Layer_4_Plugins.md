# LAYER 4: PLUGINS (THE GEAR)
**Role:** Specialty Functions.

### 📸 SCREENSHOT PROTOCOL
Capture these to ensure gear availability:
1. **Plugins -> Installed Plugins:** Capture the full list (ensure Active status is visible).
2. **Specific Plugin Settings:** 
   - Capture **SEO** (Global Settings).
   - Capture **Simple Ads** (Ad Zones).
   - Capture **SumoMe** (Badge placement).

### 🛠️ EXTRACTION POINTS
- **Essential List:** `myEventON`, `SSL Zen`, `Mashshare`, `Rich Snippets`, `SumoMe`.
- **API Keys / Licenses:** Ensure you have these saved for paid plugins.

### 📥 INJECTION
1. Go to **Plugins -> Add New**.
2. Install the list one by one.
3. Configure settings using your "Specific Plugin Settings" screenshots.
