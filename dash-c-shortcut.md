# —c  (Mobile Text Expansion Shortcut)

## What is this?

Type `—c` on your phone keyboard and it expands into the full prompt below.
Set it up once in your phone's text replacement settings, then never type a setup message again.

---

## The Shortcut Text (copy everything below the line)

---

I'm jumping back into a project. Here's my context:

**Repo:** github.com/cdowns78999/master-Central-database

**Subfolder:** [which folder]

**What I'm working on:** [one sentence — what feature/fix/task]

**Where I left off:** [last thing done or last file touched]

Pick up exactly where I left off. Read the relevant files, summarize the current state in 3 bullets, then ask me what to do next.

---

## How to set it up

### iPhone
1. Settings > General > Keyboard > Text Replacement
2. Tap **+**
3. **Phrase:** paste the full shortcut text above
4. **Shortcut:** `—c`
5. Save

### Android (Gboard)
1. Settings > System > Language & Input > Personal Dictionary
2. Tap **+**
3. Paste the full shortcut text into the top field
4. Type `—c` in the shortcut field
5. Save

---

## How it works

1. Open any chat with Claude
2. Type `—c` — it expands into the full prompt
3. Fill in the 3 bracketed fields (subfolder, task, last stop)
4. Send — coding starts immediately, zero friction
