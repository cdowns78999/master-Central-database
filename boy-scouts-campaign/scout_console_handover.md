## 🚀 The Clever Design Mechanism
The site is built as a **Single Page Application (SPA)** that functions like a professional console/command center.
- **App Launcher**: A sidebar (navigation) that swaps content dynamically without page reloads.
- **Themed Design System**: 5 distinct, sophisticated visual options (Tactical, Canopy, Heritage, Lodge, Neon) controllable via a Theme Picker HUD.
- **Data-Driven**: All major content is pulled from a centralized `data.json`.
