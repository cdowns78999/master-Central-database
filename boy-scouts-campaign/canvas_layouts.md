# Scout Console - Canvas Layouts

This document contains 5 sophisticated text-only "canvas" layouts that visually demonstrate the connections between different Boy Scout systems. These can be used as blueprints for UI sections or presentation slides.

---

## 1. Rank Backbone Canvas

```text
[ SCOUT JOINS ]
      |
      v
[ HANDBOOK ISSUED ]
  - Shows rank path
  - Lists basic gear & uniform notes
      |
      v
[ TROOP ACTIVITIES ]
  - Weekly meetings
  - Camping trips
  - Service projects
      |
      v
[ REQUIREMENTS DONE ]
  - Rank skills signed in handbook
  - Nights of camping logged
      |
      v
[ REVIEW & BOARD ]
  - Scoutmaster conference
  - Board of review
      |
      v
[ RANK AWARDED ]
  - Patch added to uniform
  - Announced at court of honor
      |
      v
[ NEXT RANK GOAL SET ]
  - Scout checks handbook again
  - Chooses new focus areas
```

## 2. “Four Systems, One Scout” Canvas

```text
                  [ SCOUT ]
                     |
  ------------------------------------------------
  |                 |               |            |
  v                 v               v            v

[ RANK TRACK ]   [ MERIT BADGES ]  [ CAMPING ]  [ UNIFORM ]
- Scout →        - Elective &      - Monthly    - Rank patch
  Tenderfoot →     Eagle‑req.        trips      - Merit badges
  ... → Eagle    - Counselors      - Summer     - Position patch
                 - Blue cards        camp       - Event patches

  |                 |               |            |
  |                 |               |            |
  ------------------+---------------+------------
                     v
             [ HANDBOOK / GUIDE ]
             - Shows how rank, badges,
               and camping fit together
             - Checklists & sign‑offs
```

## 3. Planning → Event → Recognition Canvas

```text
[ HANDBOOK + LEADER GUIDE ]
  - Official requirements
  - Program themes
        |
        v
[ ANNUAL / MONTHLY PLAN ]
  - Pick focus ranks & badges
  - Schedule campouts & meetings
        |
        v
[ MEETINGS & CAMPOUTS ]
  - Teach skills
  - Run requirements
  - Log camping nights
        |
        v
[ ADVANCEMENT RECORDS ]
  - Handbook sign‑offs
  - Badge completions
  - Nights & service tracked
        |
        v
[ COURT OF HONOR ]
  - Present rank
  - Present merit badges
        |
        v
[ UNIFORM UPDATED ]
  - New patches sewn on
  - Sash / shirt now reflects progress
```

## 4. “Uniform as Dashboard” Canvas

```text
                [ UNIFORM = LIVE STATUS ]

[ LEFT POCKET ]
- Current rank badge
- Scout’s place on path

[ MERIT BADGE SASH ]
- Row of earned skills
- Eagle‑required highlighted

[ RIGHT POCKET / TEMP PATCHES ]
- Campouts
- Special events
- High‑adventure

[ SHOULDER AREA ]
- Position patch
- Patrol emblem
- Troop numbers

[ COLLAR / NECKERCHIEF ]
- Identity with troop
- Council / camp branding

      ^
      |
      +------------------------------+
      |                              |
[ HANDBOOK / RECORDS ]        [ EXPERIENCES ]
- What should be here       - Meetings, campouts,
- Checklists & dates          service, leadership
```

## 5. Parent/Scout “What’s Next?” Canvas

```text
[ START: "WHAT SHOULD I WORK ON NEXT?" ]
                  |
                  v
     [ CHECK HANDBOOK / APP ]
     - Next rank requirements
     - Open merit badges
                  |
                  v
     [ CHECK TROOP CALENDAR ]
     - Upcoming campouts
     - Meeting themes
     - Service days
                  |
                  v
     [ MATCH NEEDS TO EVENTS ]
     - Need camping nights? → Sign up for campout
     - Need cooking reqs?  → Go to cooking night
                  |
                  v
     [ DO THE THING ]
     - Attend event
     - Practice skill
     - Ask for sign‑off
                  |
                  v
     [ UPDATE & SHOW IT ]
     - Handbook signed
     - Online records updated
     - Plan when patch will go
       on uniform
                  |
                  v
[ NEW STATUS CHECK ]
"Okay, now what’s the next step?"
```
