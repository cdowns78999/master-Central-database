# 🎬 RAVERACTING OPS DASHBOARD | SLOT 6
## OBJECTIVE: BLOG REVIVAL & TALENT PIPELINE

This system is designed as a **Central Nervous System** for `raveracting.com`.

---

## 🏗️ CORE PILLARS

### 1. The "Wing" Navigation
- Permanent left-aligned nav for rapid switching between ops modes.
- Visual feedback on active tabs using the "Cyan Glow."

### 2. Talent Recruitment (The Grid)
- Left Column: Active talent cards with follower counts and genres.
- Right Column: Pipeline tracker for outreach status.
- Goal: Move talent from "Prospect" to "Potential Job."

### 3. Content Curation (The Feed)
- Multi-channel input (Festivals, Artists, Tech, Culture).
- Automated draft generation trigger (Visual placeholder).
- Feed hierarchy: Category -> Platform Source -> Story Clip.

---

## 🛠️ TECHNICAL STACK
- **Engine**: Single-page application logic via `screens.js`.
- **Aesthetic**: Glassmorphism with a deep "Void" background (`#060112`) and neon accents.
- **Form Factor**: Desktop-first "Wing" layout for production efficiency.
