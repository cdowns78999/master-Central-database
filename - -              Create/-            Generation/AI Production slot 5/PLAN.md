# Premium Bike Shop Development Plan 🚲

This project is a high-end e-commerce mobile app inspired by Peugeot design and your specific asset sheet.

## 🎨 Asset DNA
- **Main BG:** `#242C3B`
- **Highlight:** `#37B6E9` (Light Blue)
- **Active State:** `#4B4CED` (Primary Glow)
- **Secondary:** `#353F54`
- **Deepest:** `#222834`

## 🏗️ Folder Map
- **`index.html`**: The iOS/Chrome Shell.
- **`style.css`**: The color-locked style guide.
- **`data.json`**: Bike inventory (Peugeot, Raleigh, Carbon Fiber models).
- **`/screens/`**:
  - `home.html`: Featured bikes with categorical filters (Road, Mountain, etc.).
  - `detail.html`: Full specs page with "Add to Cart".
  - `cart.html`: Checkout experience.

---
## 🛠️ Performance Protocol
1. **Z-Space:** We use multiple blur layers for the "floating" feeling.
2. **Modular:** We code each screen to match the Figma logic individually.
3. **PWA:** Ready for "Add to Home Screen" on iOS with no white-flashing.
