15–45 minutes depending on depth and breadth of the research question.

Freshly organized vault with new MOC (Maps of Content), linked references, and a CHANGES.md log showing exactly what was reorganized, linked, or updated.
Estimated Time
20–60 minutes depending on vault size. Works great with hundreds or thousands of notes.

05
The PR Machine
Runs continuous-claude overnight. Creates branches, writes code, pushes PRs, waits for CI, merges on green, repeats.
6-12 hours
continuous-claude GitHub Actions CI/CD
Setup
Install continuous-claude. Point at your repo. Define SHARED_TASK_NOTES.md with your refactoring goals and priorities. Start Friday night for a full weekend of autonomous coding.
You Wake Up To
5–20 merged PRs. Full refactor complete. CI all green. Each PR is atomic, well-documented, and tested. Monday morning you're sprints ahead.
Estimated Time
6–12 hours of continuous autonomous operation. Runs all night, creating and merging PRs in a loop.
~$50-100 API costs per weekend run

The Compound Machine.

08
The Swarm
12 Claude agents working in parallel on your codebase. One refactors, one tests, one documents, one optimizes. Coordinated via claude-flow.
6-10 hours
claude-flow ccswarm Git worktrees CI/CD
Setup
Install claude-flow. Define agent roles in YAML — refactorer, tester, documenter, optimizer, reviewer, etc. Create a task backlog with priorities. Launch the swarm at 10PM and go to sleep.
You Wake Up To
10,000+ lines of coordinated changes. New features built. Tests passing. Docs updated. Performance optimized. All agents worked in isolated worktrees and merged cleanly.
Estimated Time
6–10 hours. The agents coordinate in parallel, so total throughput is 12x a single agent session.
~$150-300 per swarm session

The Autopilot
Full business automation. Responds to leads, processes invoices, publishes content, updates CRM, monitors competitors — all overnight.
24/7 Continuous
Sleepless Agent Slack n8n Claude API

Continuous 24/7 operation. This isn't a one-off routine — it's a permanent automation layer for your business.
