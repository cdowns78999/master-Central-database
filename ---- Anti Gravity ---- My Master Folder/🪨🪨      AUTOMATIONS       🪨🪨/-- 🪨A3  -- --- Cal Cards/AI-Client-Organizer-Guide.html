04 — The Prompt

Master client sort prompt

Copy this and run it in Claude whenever you want to process a batch of messages.

Read every file in my drop folder. For each conversation, extract:

• Client name (or best guess from context)
• Channel (text, messenger, email, DM, etc.)
• What they want — summarize in 1 sentence
• Status — New Lead / Active / Follow Up Needed / Cold / Done
• Urgency — Low / Medium / High / ASAP
• Last contact date if visible
• Any $ amounts or project scope mentioned

Output this as a clean table sorted by Urgency (ASAP first). Then give me a separate section with any action items I need to handle today.
⚡
Add this as a custom Claude skill so you can trigger it with /sort-clients anytime. Just drop new messages in the folder and run the skill — instant processing.
05 — Claude Skill

Turn it into a /sort-clients skill

One-time setup so you never have to think about the prompt again.

1

Create the skill folder

In your Claude skills directory, create a folder called sort-clients with a skill.md inside it.

C:\Users\chad\.claude\skills\sort-clients\skill.md
2

Write the skill

Paste the master sort prompt into the skill instructions. Include the drop folder path, output format, and where to save results.

3

Run /sort-clients

Next time you have a batch of conversations ready, type /sort-clients in Claude Code. Claude reads the drop folder, sorts everything, and saves the results.

4

Optional: auto-output to HTML card

Chain it with your /go workspace — the skill can write the sorted client table directly into work style 1.html so you see it instantly in the browser.
