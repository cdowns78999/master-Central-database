20 Patterns the Community Is Using Right Now

① BENTO

Bento Grid

Modular tiles in varied sizes — the Apple / Linear / Vercel standard for 2025–26.

For music: one tile = latest release, one = upcoming show, one = press, one = streams-to-date.

Pattern to stealMix 1×1, 2×1, and 2×2 tiles with rounded corners & soft shadow. Warm off-white card bg, not pure white.

② EDITORIAL

Magazine Column Layout

Serif headlines, generous whitespace, small drop-caps — Vogue / Kinfolk energy.

Artist bio pages, release stories, tour diaries.

Pattern to stealTwo-column with an asymmetric image break. Line-height 1.6, 15–16px body, serif heads.

③ TAP-REVEAL

Tap-to-Reveal Tile Stack

Cards that stay quiet until you tap — reveal lyrics, notes, or memories underneath.

Lyric journals, fan letters, behind-the-scenes per song.

Pattern to stealFront face = title only, soft fade. Tap flips or slides card — keep reveal under 250ms.

④ SOFT-FILTERS

Mood-Chip Filters (not genre dropdowns)

Filter music by feeling — "golden hour," "driving home," "cozy," "heartbreak."

Huge win for women experiencing a music app — vibes over taxonomy.

Pattern to stealRounded pill chips, multi-select, active state = warm fill not harsh border.

⑤ TIMELINE

Vertical Release Timeline

Anchored by year, scroll down = history; up = recent — ribboned and illustrated.

Discography pages, tour history, label milestones.

Pattern to stealLeft rail with sticky year labels, right side = cards. Animate rail dot on scroll.

⑥ MOODBOARD

Scrapbook / Moodboard Collage

Irregular photo grid, slight rotation, pinned-note feel — Pinterest & Are.na hybrid.

Press kits, album aesthetics, visual identity pages.

Pattern to stealCSS masonry (column-count: 3) + a tiny rotation (-2deg to +2deg) per card.

⑦ LINKTREE+

Vertical Card Stack ("Linktree elevated")

One column of warm dividers, each link = mini card with icon + subtitle.

Artist "hub" landing pages replacing stiff Linktree defaults.

Pattern to stealFull-width cards, 64px tall, left icon + 2-line text, accent underline on hover.

⑧ CAROUSEL

Curved-Edge Horizontal Carousel

Snap-scroll shelves — Spotify / Apple Music row pattern but bigger, softer corners.

"New from your artists," "made for you," "weekly picks" rails.

Pattern to stealscroll-snap-type: x mandatory + 24px gap + peek of next card on the right edge.

⑨ STICKER

Sticker-Sheet Layout

Cards styled like peelable stickers — soft shadow, white border, playful corners.

Collectible "era" tiles, fan rewards, achievements.

Pattern to steal2–4px white inner border + drop shadow + slight hover lift to simulate peel.

⑩ POLAROID

Polaroid Artist Roster

Each artist as a Polaroid — handwritten name tag, slight tilt, pinned-corner feel.

Label rosters, tour lineups, collaborator pages.

Pattern to stealWhite frame with 40px bottom gutter, handwritten font (Caveat, Kalam) for names.

⑪ GLASS

Glassmorphism Layered Depth

Frosted blur cards over a soft gradient — elegant, feminine, quietly tech-y.

Now-playing overlays, lyric cards, donation / tip widgets.

Pattern to stealbackdrop-filter: blur(24px) + 6% white tint + 1px inner highlight.

⑫ ACCORDION

Illustrated Accordion Rows

Collapsible rows with a micro-illustration per row — playful, not corporate.

FAQ pages, gear lists, set-list archives.

Pattern to steal48–56px row height closed, soft rotate caret, illustration 32px on the left.

⑬ NOTION

Notion-Style Nested Database

Toggles, cover images, tags — information that reveals itself only when needed.

Songwriting vaults, session notes, A&R trackers.

Pattern to stealTop-level rows with a toggle caret, nested grid inside at one indent, cover images optional.

⑭ DUAL-PANE

Master-Detail Reading Layout

Left list, right detail — Gmail / Things 3 / Apple Mail quiet sophistication.

Inbox for DMs from labels, song revision threads, contract previews.

Pattern to stealLeft col 320px, right flex. Active row = warm fill + left accent bar.

⑮ RIBBON

Ribbon & Banner Headers

Section dividers shaped like ribbons with cursive script — boutique-brand feel.

Chapter markers in long-form bios, "new release" badges, tour posters.

Pattern to stealSVG ribbon shape + script font (Pinyon, Allura) at 32px, warm accent fill.

⑯ TOC

Sidebar Table-of-Contents with Glow

Sticky TOC on the side, active section underline glows warm as you scroll.

Long press pages, artist bios, label manifestos.

Pattern to stealIntersectionObserver to toggle active state, warm 2px left border on active.

⑰ WHEEL

Radial / Wheel Navigation

Rare but striking — circular menu, tap a wedge to drill in. Feels like a vinyl spinning.

Genre explorers, era selectors, mood dials.

Pattern to stealConic gradient background, 6–8 wedges max, center = current selection label.

⑱ ONBOARD

Step-Tracked Onboarding with Confetti

Progress ring + tiny celebration micro-moments per completed step.

Artist setup, fan club signup, first-playlist-building flow.

Pattern to stealSVG progress ring + confetti burst on step complete. Keep confetti under 600ms, warm palette.

⑲ PEEK

Expandable "Peek" Preview Cards

Hover or long-press reveals a preview before you fully commit to the tap.

Song previews (15s waveform peek), venue previews, merch zoom.

Pattern to steal400ms long-press → scale card 1.04 + reveal preview layer. Release = collapse.

⑳ KANBAN

Swim-Lane Kanban (soft edition)

Columns labeled by stage — but softened, warm, feminine. Trello stripped of its blue.

Release pipeline (Writing → Recording → Mixing → Release → Promo).

Pattern to stealWarm off-white columns, drag handles only on hover, soft 8px radius cards.
