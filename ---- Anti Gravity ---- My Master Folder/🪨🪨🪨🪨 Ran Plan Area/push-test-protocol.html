Push Notification Test Protocol — Wing Dashboard

Step-by-step · Windows · Android · iOS

🪟

Windows

Chrome Desktop

Progress 0 / 6
01 Make sure push-server.py is running — open PowerShell, run it, leave the window open
02 Open https://thennow-homebase.netlify.app in Chrome
03 Click 🔔 Enable Notifications → click Allow when the browser permission prompt appears
04 Console should log "FCM Token saved" — open DevTools (F12) → Console tab to verify
05 Go to Wing Dashboard (open local index.html) → find the THENNOW client card → click 🔔 Send Push
06 Windows notification pops in the bottom-right corner ✅
🤖

Android

Chrome Mobile

Progress 0 / 5
01 Open https://thennow-homebase.netlify.app in Chrome on your Android device
02 Tap 🔔 Enable Notifications → tap Allow on the permission prompt
03 Token saves automatically to thennow.json on the server — no manual step needed
04 From Wing Dashboard on your computer → click 🔔 Send Push on the THENNOW card
05 Android notification drops into the notification tray ✅
🍎

iOS

Safari — requires PWA install

Progress 0 / 6
01 Open https://thennow-homebase.netlify.app in Safari on your iPhone
02 Tap the Share button (box with arrow ↑) → tap "Add to Home Screen" → tap Add
03 Close Safari completely — then open the app from your Home Screen icon (must launch from icon, not Safari)
04 Tap 🔔 Enable Notifications → tap Allow on the iOS permission prompt
05 From Wing Dashboard on your computer → click 🔔 Send Push on the THENNOW card
06 iOS notification appears on lock screen / notification center ✅
push-server.py must be running on port 3005 for all three tests to work
