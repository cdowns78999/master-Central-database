# Event Run-of-Show — VR Speed Dating Night

A weekly 90-minute event. Fridays at 8:00 PM ET (adjust per timezone polling).

---

## The arc

```
  [raffle]   ->   [invite drop]   ->   [lobby warm-up]   ->   [5 rotations x 2 min]   ->   [post-event reveal]
  T-48h           T-30min              T-10min              event window              next morning
```

---

## T-48 hours · Wednesday

**Raffle opens in Discord.**
- Host posts two GiveawayBot giveaways in `#raffle-room`: `Group A (15)` and `Group B (15)`.
- Open to `@Table` patrons. Front Row patrons already pre-seated (10 guaranteed).
- Pinned message includes theme, dress code hint, and "if you win but can't make it, please un-claim by Thursday 8 PM."

---

## T-24 hours · Thursday

**Host balancing pass.**
- GiveawayBot auto-closes, picks 30 random entrants.
- Host pulls winner list into the onboarding-form spreadsheet.
- Manually reassigns as needed to achieve 15+15 balance (orientation, intent, tag variety, age spread).
- Posts: *"Group A + Group B confirmed — see you Friday."*

---

## T-30 minutes · Friday 7:30 PM

**Host drops the invite link.**
- Host creates nonce-locked VRChat instance in **La Barista** (or Cafe Vismo once commissioned).
- Drops the `vrchat://` deep-link in `#event-invites` channel (Table+ visible).
- Pinned intro post reminds:
  - Mic check in Cafe Patio voice channel
  - "Don't be late — rotations start at :00 sharp"
  - Code of conduct reminder

---

## T-10 minutes · 7:50 PM

**Lobby warm-up in VRChat.**
- Patrons spawn in the cafe foyer.
- Host is visible wearing an emcee badge avatar accessory.
- Soft French cafe music (Edith Piaf / instrumental jazz) playing from the world's audio source.
- Nameplates show: **Pronouns · 2 of 5 tags**. (Rest revealed during rotations.)
- Host calls a quick "say your Discord handle in voice" round so everyone's mic is confirmed.

---

## 8:00 PM — Rotations begin

**Format:** 5 rotations × 2 minutes each + 30 seconds transition.

Total event window: **12.5 minutes of pairings** (plus framing) = ~30 minute active dating window.

| Rotation | Time         | What happens                                           |
|----------|--------------|--------------------------------------------------------|
| Intro    | 8:00 – 8:02  | Host gives the evening's conversation prompt (see below) |
| R1       | 8:02 – 8:04  | Pair 1 at each table (2-min timer on the ceiling clock) |
| Shuffle  | 8:04 – 8:04:30 | Host rings the bell, Group A rotates one table over  |
| R2       | 8:04:30 – 8:06:30 | Pair 2                                            |
| Shuffle  | 8:06:30 – 8:07 | Bell                                                 |
| R3       | 8:07 – 8:09  | Pair 3                                                 |
| Shuffle  | 8:09 – 8:09:30 | Bell                                                 |
| R4       | 8:09:30 – 8:11:30 | Pair 4                                            |
| Shuffle  | 8:11:30 – 8:12 | Bell                                                 |
| R5       | 8:12 – 8:14  | Pair 5                                                 |
| Close    | 8:14 – 8:30  | Hangout in the cafe patio, host open Q&A              |

---

## Conversation prompts (host seeds one per rotation)

A prompt is NOT a script. It's a spark. Host can skip a prompt if the round is already flowing.

**Rotation 1:** *"What brought you to VR in the first place?"*
**Rotation 2:** *"What's something you'd do every week if time and money weren't factors?"*
**Rotation 3:** *"Most embarrassing song in your current rotation?"*
**Rotation 4:** *"One tag on your profile you almost didn't pick — why did you?"*
**Rotation 5:** *"What's a small thing that made you laugh this week?"*

Host tier patrons can propose prompts in `#host-green-room` the week before.

---

## Post-event · Reveal DM flow

**That night, within 2 hours of the event:**
1. Host posts a short web form (Tally) link in `#event-invites` DM-style.
2. Form: *"For each of your 5 rotation partners, choose:* ✨ *yes, connect me* · 🤝 *friend only* · 👋 *no thanks (no hard feelings)."*
3. Form closes Saturday midnight.

**Saturday morning:**
- Chad + Vismo run the match script (manual for MVP, automated later).
- For each **mutual yes**: both parties get a DM with the other's Discord handle.
- For each **mutual friend**: both parties get a DM with friend-emoji + handle.
- Non-mutual: no notification (rejection is silent).
- Aggregate stats post in `#mutual-matches`: *"This week: 30 patrons, 47 rotations completed, 8 mutual connects, 5 mutual friends."* No names.
