VR Bridge Install Flow

Voice translator + auto-invite-on-online pipeline for VRChat. Follow the 6 steps top to bottom.

  +-------+    +-----+    +-----------+    +-----+    +----------+    +--------+
  |  mic  | -> | STT | -> | translate | -> | TTS | -> | VB-Cable | -> | VRChat |
  +-------+    +-----+    +-----------+    +-----+    +----------+    +--------+

  +------------+    +---------------+    +---------------+
  | VRChat API | -> | invite-bot.js | -> | group.invite  |
  +------------+    +---------------+    +---------------+
1 Install VB-Cable (Virtual Audio Cable)
vb-audio.com/Cable
  • Creates a fake microphone/speaker pair so apps can pipe audio to each other.
  • Install, reboot, confirm CABLE Input and CABLE Output appear in Windows Sound settings.
  • Route: VR Bridge TTS output sends to CABLE Input, then VRChat reads from CABLE Output as its mic.
2 Install Voicemeeter Banana
vb-audio.com/Voicemeeter
  • Acts as a software mixer so you can monitor and blend your real mic with the translated voice.
  • Hardware Input 1 = your real mic. Virtual Input (VAIO) = CABLE Output.
  • A1 output = your headphones (for monitoring). B1 output = VRChat mic bus.
  • In VRChat audio settings, set input device to Voicemeeter Out B1.
3 Get OpenAI API key (free $5 trial)
platform.openai.com/api-keys
  • Powers Whisper STT (speech-to-text) and GPT translation.
  • Click Create new secret key, copy it once (you won't see it again).
  • Paste into .env:
OPENAI_API_KEY=sk-...
4 Get ElevenLabs API key (10k free chars/mo)
elevenlabs.io
  • High-quality TTS voice for the translated output.
  • Sign up, open Profile → API Key, copy the key.
  • Paste into .env:
ELEVENLABS_API_KEY=...
5 VRChat login + Group ID
  • Open your VRChat group in the browser. The URL looks like vrchat.com/home/group/grp_abc123... — the grp_... portion is your Group ID.
  • Find each friend's User ID (starts with usr_...) from their profile URL, comma-separate them for the watch list.
  • Add to .env:
VRCHAT_USERNAME=your_vrc_username
VRCHAT_PASSWORD=your_vrc_password
VRCHAT_GROUP_ID=grp_abc123-...
WATCH_FRIEND_IDS=usr_111...,usr_222...,usr_333...

Note: 2FA-enabled accounts will prompt for a code the first time the bot logs in.

6 Launch

In the project folder:

npm install
npm start

Then open http://localhost:3000 to use the voice translator UI.

For the auto-invite-on-online watcher, open a second terminal and run:

npm run invite-bot

Keep both running together. The invite bot polls friend presence and invites them to your group when they come online.

⚠️ Ban Risk Disclosure

Automation uses the unofficial VRChat API. VRChat does not publicly support third-party automation of invites, friend actions, or presence polling.

This tool is rate-limited to 1 invite per friend per hour to stay conservative, but any automation carries risk.

Use at your own risk. It can result in account warnings, suspension, or a full ban. By running VR Bridge, you accept this risk.
