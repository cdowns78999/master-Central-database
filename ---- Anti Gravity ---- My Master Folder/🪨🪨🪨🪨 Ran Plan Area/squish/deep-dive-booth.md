## 1. How BOOTH.PM Works

### What Is It?

BOOTH.PM is a Japanese indie marketplace run by Pixiv. It is the dominant third-party platform for buying and selling VRChat avatar bases, clothing, accessories, and other 3D assets. It is separate from VRChat's own official Avatar Marketplace.

- **URL:** https://booth.pm/en (English interface available)
- **Creator cut:** Approximately 96.4% of every sale goes to the creator (one of the most creator-friendly rates anywhere)
- **Currency:** All prices are in Japanese Yen (JPY). As of 2026, the weak yen makes purchases very favorable for USD/EUR buyers
- **Inventory:** 38,000+ VRChat avatar-related 3D models; 3,198+ avatar bases specifically; 7,299+ free avatar items

### Account Setup

1. You need a **Pixiv account** (free) to use BOOTH
2. Go to https://booth.pm/en and click "Sign In" — it redirects to Pixiv account creation
3. Registration takes under 5 minutes; no Japanese address required for digital downloads
4. Set your preferred language to English in account settings

### Payment Methods

- **Credit cards:** VISA, Mastercard, JCB accepted directly
- **PayPal:** Was previously supported but has been **temporarily suspended** as of March 2024. Status may have changed — check booth.pm/payment for current info
- **Debit cards:** Work if registered through PayPal (when available)
- **Important:** American Express is NOT directly supported
- **Note:** For digital downloads (which all avatar bases are), no shipping/proxy service is needed. You download directly after purchase

### How Buying Works

1. Find an avatar or asset on BOOTH (or via BOOTHPLORER)
2. Add to cart and checkout
3. Pay in JPY (your bank handles conversion automatically)
4. Download the .zip file containing Unity packages, textures, documentation
5. Import into Unity, customize, upload to VRChat

### Language Barrier

- Most sellers are Japanese creators who may not speak English
- Product descriptions are often in Japanese only, though bilingual (JP/EN/KR) shops are increasing
- Use browser translation (Chrome auto-translate works well on BOOTH)
- Many avatar pages include visual guides that transcend language

## 3. Purchase-to-Upload Workflow

### Prerequisites

Before doing anything:

1. **Install VRChat Creator Companion (VCC):** Download from https://vrchat.com/home/download
2. **Unity:** VCC auto-installs Unity 2022.3.22f1 (the required version). Do NOT use a different Unity version
3. **VRChat Account:** You need a VRChat account to upload avatars

### Step-by-Step Process

#### Step 1: Create a New Avatar Project
1. Open VCC
2. Click "New" in the left nav
3. Choose "Avatar" template
4. Name your project and hit "Create New Project"
5. VCC creates a Unity project pre-configured with Avatar 3.0 SDK

#### Step 2: Install Required Shaders
Most Booth avatars use one of two shaders. Install these BEFORE importing the avatar:

- **Poiyomi Toon Shader** — Free, open-source, feature-rich. Get from booth.pm or github.com/poiyomi/PoiyomiToonShader
- **lilToon** — Another popular free shader. Available via VCC package manager

Check the avatar's product page or included README to see which shader it uses.

#### Step 3: Import the Avatar Package
1. Extract the downloaded .zip from BOOTH
2. Look for `.unitypackage` files inside
3. In Unity: Assets > Import Package > Custom Package
4. Or simply drag the `.unitypackage` file into the Unity project window
5. Click "Import" when the import dialog appears

**Important:** If the package contains VRCSDK folders, UNCHECK them during import. The VCC already installed the SDK — importing a second copy causes conflicts.

#### Step 4: Set Up the Scene
1. Find the avatar prefab in your Project panel (usually in an Assets subfolder named after the avatar)
2. Drag the prefab into the Hierarchy panel
3. The avatar should appear in the Scene view

#### Step 5: Configure and Upload
1. Open VRChat SDK Control Panel (VRChat SDK > Show Control Panel)
2. Log into your VRChat account
3. Select your avatar in the scene
4. Add a name, description, and thumbnail image
5. Click "Build & Publish for Windows" (or Android for Quest)
6. Accept the terms and upload

#### Step 6: Find It In-Game
1. Launch VRChat
2. Open the Avatar menu
3. Go to "My Avatars"
4. Your uploaded avatar should appear there

## 4. Clothing & Accessory Ecosystem

### How Compatibility Works

VRChat clothing is NOT universal. Each clothing item is designed for specific avatar bases. When browsing BOOTH:

- Look for compatibility tags like "対応: Kikyo, Shinra, Manuka" (compatible with: ...)
- Clothing items typically list all supported avatars in the product title or description
- Popular avatars like Kikyo and Shinra have 9,000-10,000+ compatible items
- Less popular bases may only have a few hundred

### What "Compatible" Means

A compatible clothing item includes:
- Pre-rigged meshes that match the avatar's bone structure
- Blendshapes that align with the avatar's body morphs
- Materials/textures configured for the avatar's UV layout
- Usually a prefab ready for drag-and-drop

### How to Add Clothing

#### Method 1: Modular Avatar (Recommended)
1. Install Modular Avatar via VCC
2. Drag the outfit prefab onto your avatar in the Hierarchy
3. Right-click the outfit > "ModularAvatar > Setup Outfit"
4. Modular Avatar auto-detects the armature and merges it
5. Upload — done

Key advantage: **Non-destructive**. You don't need to unpack the original avatar prefab, making future updates easy.

#### Method 2: VRCFury
1. Install VRCFury (free Unity plugin)
2. Similar drag-and-drop workflow
3. Automatically merges rigged clothing, adds toggles to VRChat menu, handles animation controllers
4. Also non-destructive

#### Method 3: Manual (Advanced)
1. Remove existing outfit meshes if needed
2. Import the clothing .unitypackage
3. Manually parent the clothing to the avatar's armature
4. Merge bone hierarchies by hand
5. Set up animation toggles manually

### Types of Compatible Items

- **Full outfits** — Complete clothing sets (dresses, suits, uniforms)
- **Individual pieces** — Tops, bottoms, shoes, socks, gloves
- **Hair** — Alternative hairstyles rigged to the avatar's head
- **Eyes** — Texture swaps for different eye styles/colors
- **Skin textures** — Alternate skin tones, tattoos, freckles, body details
- **Accessories** — Glasses, jewelry, piercings, wings, tails, hats
- **Facial expressions** — Additional expression shape keys
- **Body morphs** — Shape key packages for body customization
- **Props** — Held items, weapons, instruments

## 8. How to Customize a Base

### Color Changes (Easy)

#### Changing Hair/Outfit Colors via Material Properties
1. Select the avatar in Unity
2. Find the hair or outfit material in the Inspector
3. If using Poiyomi or lilToon shader, look for "Main Color" or "Base Color"
4. Adjust the color picker or hue shift slider
5. Some avatars include color-change presets

#### Changing Colors via Texture Editing
1. Find the avatar's texture files (usually .png or .psd)
2. Open in Photoshop, GIMP, or Krita
3. Adjust hue/saturation on specific layers
4. Save and reimport into Unity
5. The material auto-updates

### Swapping Hair (Medium)

1. Purchase a compatible hair asset from BOOTH (search "[avatar name] hair")
2. Import the hair .unitypackage into your project
3. Using Modular Avatar: drag the hair prefab onto your avatar, right-click > Setup Outfit
4. Hide or delete the original hair mesh
5. Upload

### Adding Clothes (Medium)

1. Purchase compatible clothing
2. Import .unitypackage
3. **With Modular Avatar:**
   - Drag outfit prefab onto avatar
   - Right-click > ModularAvatar > Setup Outfit
   - Done — it auto-merges armature
4. **With VRCFury:**
   - Similar drag-and-drop
   - Auto-generates toggles for VRChat menu
5. Remove/hide original clothing if it clips

### Swapping Eyes (Easy)

1. Buy eye textures from BOOTH (very cheap, often 200-500 JPY)
2. Import the texture
3. Assign it to the eye material slot on your avatar
4. Adjust settings like emission for glow effects

### Adding Accessories (Medium)

1. Import accessory .unitypackage
2. Drag onto appropriate bone in the avatar hierarchy (e.g., "Head" bone for glasses)
3. Position in Scene view
4. Most accessories come with Modular Avatar or VRCFury setup already configured

### Body Shape Customization (Easy-Medium)

1. Select the avatar's SkinnedMeshRenderer in Inspector
2. Scroll to BlendShapes section
3. Adjust sliders for body proportions (bust, waist, hips, height, muscle, etc.)
4. Changes are real-time in the editor

### Adding Tattoos/Skin Details (Easy)

1. Purchase skin texture overlays from BOOTH
2. Many use decal systems that layer over existing textures
3. Some use emission maps for glow effects
4. Import and assign to appropriate material slots

## 10. Furry vs Anime vs Western Style

### Anime Style (Dominant on BOOTH)

- **Market share:** ~70-80% of BOOTH avatar bases
- **Characteristics:** Large eyes, small nose/mouth, colorful hair, slim builds
- **Top examples:** Kikyo, Shinra, Selestia, Manuka, Lime
- **Clothing ecosystem:** Massive — thousands of items per popular base
- **Origin:** Almost entirely Japanese creators
- **Subcategory — Kemonomimi:** Anime bases with animal ears/tails (cat ears, fox tails). Very popular. Not considered "furry" — it's a distinct aesthetic

### Furry/Anthro Style

- **Market share:** Smaller on BOOTH, larger on Gumroad/Jinxxy/VRCArena
- **Characteristics:** Animal-based body with humanoid posture, fur textures, muzzles/snouts
- **Top examples:**
  - **Novabeast** (Kittomatic) — Dragon-like with oversized manes, one of the most ubiquitous furry avatars in all of VRChat
  - **Rexouium** — Popular protogen/dragon base
  - **Wickerbeast** — Classic furry base
  - **Various species bases** on VRCArena
- **Clothing ecosystem:** Growing but smaller than anime. Furry clothing tends to be species-specific
- **Key marketplaces:** VRCArena (best index), Jinxxy, Gumroad
- **Community tool:** TierMaker has active VRChat furry base tier lists

### Western/Realistic Style

- **Market share:** Smallest segment
- **Characteristics:** More realistic proportions, Western animation influence, fewer anime tropes
- **Top examples:**
  - **ERIN / REVERB** (Battle Bunny Mods) — Male bases with realistic proportions
  - **Kiyo Base** — Western booth-styled base
  - **Rusk** — Semi-realistic anime/Western hybrid
- **Clothing ecosystem:** Limited compared to anime, growing
- **Where to find:** Gumroad, VRCMods, some BOOTH sellers

### The Blurry Lines

The boundaries between styles aren't sharp. "Kemonomimi" (animal-eared anime characters) sit between anime and furry. Some avatars blend Western proportions with anime shading. The community generally doesn't gatekeep style categories — use what you like.

## 11. Quest-Compatible Bases

### Why Quest Matters

Quest (now Meta Quest) users and VRChat Mobile users can only see avatars that meet strict performance requirements. If your avatar isn't Quest-compatible, Quest users see a fallback/robot avatar instead of you.

### Quest Performance Requirements

| Metric | Good | Medium | Poor | Very Poor (Blocked) |
|--------|------|--------|------|---------------------|
| Triangles | < 7,500 | < 10,000 | < 15,000 | > 20,000 |
| Materials | 1 | 1 | 2 | > 4 |
| Mesh Renderers | 1 | 1 | 2 | > 2 |
| Texture Size | 1024x1024 max recommended | | | |
| File Size (compressed) | | | | > 10MB = cannot upload |

- Mobile default display setting is "Medium" — anything "Poor" or worse is hidden by default
- Aim for under 15,000 triangles and 1-2 materials
- Use 1024x1024 textures maximum
- One skinned mesh renderer only

### Quest-Ready Bases

- **ERIN** (Battle Bunny Mods) — Explicitly PC + Quest versions included
- Search BOOTH for "quest compatible" — 1,205+ items tagged
- BOOTHPLORER has a mobile support filter
- Most popular anime bases (Kikyo, Shinra, Selestia) are PC-only and exceed Quest limits by far (Selestia is 107K triangles)

### Making a PC Base Quest-Compatible

If you bought a PC-only base and want Quest support:

1. **VRCQuestTools** — Free Unity editor extension (github.com/kurotu/VRCQuestTools) that helps convert PC avatars for Quest
2. **Manual optimization in Blender:**
   - Decimate mesh to under 15K triangles
   - Merge all materials into 1-2 atlas textures
   - Remove PhysBone chains (Quest doesn't support many)
   - Bake lighting into textures
3. Upload as a separate Quest version of the same avatar (VRChat supports platform-specific avatar variants)

### Reality Check

Most popular BOOTH avatar bases are NOT Quest-compatible out of the box. They're designed for PC with high polygon counts and multiple materials. Quest compatibility usually requires significant work or a dedicated Quest version from the creator.

## 12. Common Import Issues

### Issue 1: Pink/Missing Materials (Shader Problems)

**Symptom:** Avatar appears pink or materials are missing
**Cause:** Required shader not installed
**Fix:**
1. Check the avatar's product page or README for shader requirements
2. Install Poiyomi Toon Shader or lilToon BEFORE importing the avatar
3. If already imported: delete the shader folder, reimport the correct version

### Issue 2: Unity Version Mismatch

**Symptom:** Import errors, missing components, SDK conflicts
**Cause:** Using wrong Unity version
**Fix:**
- ONLY use Unity 2022.3.x (the LTS version VCC installs)
- Packages from Unity 2022 only work in 2022 projects
- 70% of lighting and import issues originate from incompatible Unity versions

### Issue 3: Duplicate SDK Files

**Symptom:** Console errors about duplicate scripts/assemblies
**Cause:** The imported .unitypackage contains VRChat SDK files that conflict with your VCC-installed SDK
**Fix:**
- When importing, scroll through the file list and UNCHECK any VRCSDK-related folders
- This commonly happens with older packages made before VCC existed

### Issue 4: Console Errors (Red)

**Symptom:** Red error messages in Unity Console
**Fix:**
1. Open Console: Ctrl + Shift + C
2. Yellow warnings can usually be ignored
3. Red errors are the real problem — read the error message
4. Common causes: missing references, incompatible packages, shader compile errors
5. Try: reimporting assets, deleting Library folder and reopening project

### Issue 5: Poiyomi Shader Inspector Not Working

**Symptom:** Material inspector shows raw shader properties instead of Poiyomi's custom UI
**Fix:**
1. Delete the entire `_PoiyomiShaders` folder from Assets
2. Reimport the latest version of Poiyomi from their official source
3. Let Unity recompile

### Issue 6: Avatar T-Posing / Not Animating

**Symptom:** Avatar stuck in T-pose
**Cause:** Animation controller not assigned or incorrect rig settings
**Fix:**
1. Check that the avatar's FBX has Rig type set to "Humanoid"
2. Ensure the VRC Avatar Descriptor component is on the root object
3. Verify animation controllers are assigned in the Playable Layers section

### Issue 7: Lighting Looks Wrong

**Symptom:** Avatar looks too dark, too bright, or has baked lighting artifacts
**Cause:** Custom shaders cause 60% of lighting bake failures
**Fix:**
1. Use Unity 2022.3.x stable LTS
2. Avoid baking lighting for avatar projects
3. Adjust shader settings for realtime lighting response

### Issue 8: Japanese File Names / Encoding Errors

**Symptom:** Files with Japanese characters fail to extract or import
**Fix:**
- Extract .zip files using 7-Zip (handles Japanese encoding better than Windows default)
- On Windows, ensure system locale supports Japanese characters
- Or rename files to ASCII before importing

## 13. Best Bases for Beginners

### If You've Never Touched Unity Before

**Start with VRoid Studio (Free)**
- No Unity knowledge needed
- Visual character creator with sliders
- Export directly to VRChat format
- Learn the basics before spending money

**Then try a free BOOTH base**
- Practice the import workflow with zero financial risk
- Browse free options at https://boothplorer.com/avatars/free
- Learn to navigate Unity, import packages, and upload

### First Paid Purchase Recommendations

#### For maximum clothing options: **Kikyo** (5,500 JPY / ~$37)
- 10,882+ compatible items — you'll never run out of outfits
- Mature ecosystem with lots of community guides
- Standard anime girl aesthetic

#### For best expression system: **Selestia** (6,000 JPY / ~$40)
- 358 facial expression shape keys
- Extremely detailed customization out of the box
- Very recognizable — you'll see other Selestia users to learn from

#### For budget-conscious: **Karin** (5,000 JPY / ~$33)
- Cheapest among the popular bases
- Strong accessory ecosystem
- Face tracking add-ons available

#### For male avatars: **ERIN** (Battle Bunny Mods)
- PC + Quest compatible (rare)
- 3 outfits included
- Extensive body morphs
- Available on both BOOTH and Gumroad (pay in USD on Gumroad)

#### For furry/anthro: **Novabeast** (Kittomatic)
- One of the most recognizable furry avatars in VRChat
- Complete package: Unity package, Substance Painter file, PSD, Blender file
- Available on Gumroad (USD) and Jinxxy (PayPal)

### Beginner Workflow Tips

1. **Install Booth Companion browser extension first** — makes BOOTH browsable in your currency
2. **Install Modular Avatar in your Unity project** — makes adding clothing trivially easy
3. **Start with one avatar base** — learn it thoroughly before buying another
4. **Join Discord communities** — most popular avatar bases have Discord servers where users share tips
5. **Watch TikTok/YouTube tutorials** — search "[avatar name] VRChat setup" for visual guides
6. **Don't skip shader installation** — import shaders before the avatar package, every time
