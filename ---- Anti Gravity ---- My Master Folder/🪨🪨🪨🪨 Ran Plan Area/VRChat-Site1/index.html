📋 How to Report In-Game

1

Open the Quick Menu

Press the Menu button on your controller (or Escape on desktop). The Quick Menu opens near the offending user.

2

Select the User

Click on the user's nameplate or find them in the Social panel. Select their profile.

3

Tap "Report"

Hit the Report button. Choose the violation category: Harassment, Hate Speech, Sexual Content, Grooming, CSAM, Scamming, or Other.

4

Add Context

Write a brief description of what happened and when. Timestamps help moderators act fast.

5

Block & Mute Immediately

Don't wait for a response. Block the user right away — this removes them from your view and prevents further contact.

6

Report via Website Too

For serious violations, also submit at hello.vrchat.com — attach screenshots or video clips if possible.

Invisible shield that prevents other avatars from walking through you. Enable via Quick Menu → Safety → Bubble.

Control whose avatars you see: Everyone / Friends / Friends of Friends / None. Start with "Friends only" until comfortable.

🟣 Know the Lingo

VRChat Slang Dictionary

You stepped into a lobby and had no idea what anyone was saying. We've got you. The complete VRChat glossary.

📖 Term of the Week: Phantom Sense

The psychological phenomenon where a VR user physically feels sensations corresponding to in-game events — like feeling tapped on the shoulder when your avatar is touched. Not a glitch. Your brain is just that immersed.

🔍

Crasher

SAFETY

A user who deliberately uses malicious avatars or exploits to crash other users' VRChat clients. Crashers are bannable offenses — always report them.

EXAMPLE

"Someone just crashed the entire lobby — there's a crasher in the instance."

Newcomer Tip: Report + Leave immediately

FBT

HARDWARE

Full Body Tracking. Using hardware trackers (Vive Trackers, SlimeVR, etc.) on feet, hips, and knees so your avatar mirrors your actual body movements in full.

EXAMPLE

"Their FBT setup is insane — every step looks 1:1."

Phantom Sense

PSYCH

The sensation of physically feeling things that happen to your VR avatar. Common after extended sessions — a form of learned embodiment and presence illusion.

EXAMPLE

"I got hugged in VRChat and I literally felt it. Phantom sense is wild."

Ripping

ILLEGAL

Extracting avatar or world assets directly from VRChat's memory/files without the creator's permission. Violates VRChat ToS and copyright law. Always buy from BOOTH.

EXAMPLE

"That's a ripped avatar — check the poly count, it's clearly extracted."

Report if you see ripped assets being sold

Kitbash

CREATION

An avatar or world prop assembled from parts of multiple different source assets. A creative technique — mixing legs from one base, torso from another, head from a third.

EXAMPLE

"My avatar is a full kitbash — took me two months to get the proportions right."

UDON

DEV

VRChat's proprietary programming system for world interactivity. Visual node-graph or UdonSharp (C# syntax). Powers everything from door buttons to full game logic.

EXAMPLE

"The quiz game runs on UDON — the dev wrote the whole thing in UdonSharp."

Questie

PLATFORM

Slang for a user on a standalone Meta Quest headset. Quest users see grey robot avatars for PC avatars that aren't Quest-compatible. Not derogatory — just descriptive.

EXAMPLE

"Can your avatar fallback? There are a few questies in here."

Physbones

TECH

VRChat's native physics component for dynamic avatar bones — hair, tails, ears, clothes, breasts, ribbons. Replaced the old DynamicBones system. Touch-interactive by default.

EXAMPLE

"The hair on this avatar has beautiful physbones — it moves so naturally."

EAC

SYSTEM

Easy Anti-Cheat. VRChat implemented EAC in 2022, which ended the era of third-party client mods (most notably VRCMods). Controversial in the community.

EXAMPLE

"EAC killed all the good client mods. Still salty about it."

OSC

DEV

Open Sound Control. VRChat exposes avatar parameters via OSC, allowing external apps to control avatar states — e.g., heart rate monitors, Lovense devices, MIDI, face tracking overlays.

EXAMPLE

"They wired their heart rate monitor via OSC — their avatar's chest glows with their pulse."

VCC

DEV

VRChat Creator Companion. The official desktop app for managing VRChat SDK versions, installing community packages like Modular Avatar, and setting up Unity projects.

EXAMPLE

"Install VCC first, then let it set up Unity — it's way easier now than before."

Instance

CORE

A live session of a world. Each world can have many simultaneous instances. Instance types: Public, Friends+, Friends Only, Invite+, Invite. The instance host has special permissions.

EXAMPLE

"The public instance was full — we had to make our own."

🟢 Build VRChat

Dev Corner

From first world to UdonSharp mastery — the complete development resource for VRChat creators.

SDK 3.10.2 — Current (Feb 2026)
Unity 2022.3.22f1 — Required
UdonSharp 2.0 — Preview

Step 1 — Install VCC

Download VRChat Creator Companion (VCC). It installs and manages your Unity version and SDK automatically. Do not install Unity manually first.

Download VCC

Step 2 — Create a World Project

Open VCC → New Project → VRChat World. This spins up a Unity project with the SDK pre-installed. Build your first room with a spawn point and scene descriptor.

Step 3 — Add Udon Logic

Use the visual Udon Node Graph to add simple interactivity — buttons, doors, object spawners. No code required. Drag behaviors from the inspector panel.

Step 4 — Publish

VRChat SDK panel → Build & Publish. Add a world name, description, thumbnail image, and tags. Set visibility to Public or Private. Upload takes seconds.

UdonSharp — Hello World
using UdonSharp; using UnityEngine; using VRC.SDKBase; public class HelloWorld : UdonSharpBehaviour { void Start() { Debug.Log("Welcome to my VRChat world!"); } public override void Interact() { Debug.Log("Player interacted with this object"); } }

UdonSharp Networking

Sync variables across all players with [UdonSynced] attribute. Master client ownership model. Use RequestSerialization() carefully — it has bandwidth costs.

OSC Integration

VRChat exposes avatar parameters and triggers on ports 9000 (receive) and 9001 (send). Use Python or Node.js to build external tooling — haptic sync, real biometrics, MIDI control.

Quest Optimization

Draw call batching, lightmap baking, occlusion culling, and LOD groups. Quest worlds must stay under 64MB total asset size. Use Unity's Frame Debugger to identify bottlenecks.

Custom Shaders (HLSL)

Write custom ShaderLab/HLSL for effects not possible with standard shaders — custom depth, screen-space effects, procedural geometry. Check Quest compatibility before shipping.

UdonSharp — Synced Counter

🛠️ FBT Troubleshooting

Ensure trackers are paired via SteamVR Settings → Devices → Pair New Device. Check for firmware updates. If using Bluetooth dongles, try USB 2.0 ports instead of 3.0.

Calibrate in VRChat by standing in T-pose and pressing both triggers simultaneously. Your avatar must have correct bone hierarchy for FBT (hip, left/right feet as separate bones).

Recalibrate SlimeVR using the "Full Reset" option, not just "Quick Reset". Ensure trackers are tight on your body — loose trackers drift significantly. Re-run IMU calibration if drifting persists.

Quest avatars support FBT but need the hip/feet bones set in the VRChat Avatar Descriptor. PC-crossplay: your PC avatar needs a Quest fallback with matching bone names. Use CATS to verify bone mapping.

HaritoraX uses a companion app (HaritoraConfigurator) to output as virtual SteamVR trackers. Ensure the companion app is running BEFORE launching VRChat. Check port 5912 is not blocked by firewall.

⚙️ How VR Sync Works

VR-integrated haptic devices connect to a companion app that listens on an OSC or Bluetooth channel. The app maps avatar proximity/contact events (triggered by VRChat's OSC output or custom avatar parameters) to device intensity signals in real time. The VRChat client itself does not communicate with hardware — the bridge is always an external companion application running alongside.

2014 — The Beginning
VRChat Founded
Graham Gaylor and Jesse Joudrey found VRChat Inc. on January 16, 2014. The initial concept is a simple VR social space — nothing more. They begin building in Altspace VR before deciding to create their own platform. No one knows what it will become.
2015 — Early Development
Alpha Builds & Enthusiast Community
Alpha builds and prototype world system take shape. Oculus Share demos attract a small but passionate group of early VR enthusiasts. The foundations of the avatar upload system are being iterated on internally. A tiny community begins to form around the concept.
2016 — Beta & SDK Preview
World SDK Improvements & Creator Interest
World SDK improvements open up more creator possibilities. Growing creator interest from the indie VR community as word spreads. The pre-Steam platform foundation is laid — the team prepares for the Early Access launch that will change everything. Content creation tools mature significantly.
2017 — Early Access
Steam Early Access Launch
VRChat launches on Steam Early Access. The avatar upload system is released — anyone can upload a custom 3D character. This single decision defines everything that follows. The creator ecosystem begins forming. A small but passionate community discovers the platform.
January 2018 — The Meme Explosion
Ugandan Knuckles & the Viral Moment
The Ugandan Knuckles meme — a distorted Sonic character asking "do you know de wae?" — goes massively viral inside VRChat. The platform is suddenly flooded with new users. Concurrent user records shatter. VRChat trends globally. The mainstream media discovers virtual reality social spaces for the first time. Monthly active users multiply by 10x in 30 days.
📌 Historical Moment

The Ugandan Knuckles moment proved that emergent user-created culture — not features, not marketing — was VRChat's core value proposition. The platform never tried to manufacture it. It just happened.

2018 — Growth & Funding
Series A Funding & Rapid Expansion
VRChat raises its Series A funding round. The team expands. Platform infrastructure scales to handle the surge in traffic. World creation tools improve significantly. The first true creator economy begins to emerge as BOOTH gains Western awareness.
2019 — Post-Viral Stabilization
Community Normalization & Early Creator Economy
Community normalization after the Ugandan Knuckles wave. The initial flood of curiosity-seekers subsides, leaving behind a more dedicated user base. Early creator economy development begins as top avatar and world creators start monetizing their work via Patreon and BOOTH. Platform infrastructure investment accelerates to support long-term growth.
2020 — The Pandemic Era
COVID-19 Drives Unprecedented Growth
When the world goes into lockdown, VRChat becomes a lifeline. People hold funerals, weddings, concerts, graduations, and therapy sessions in VRChat. Active users surge. The platform hosts its first large-scale virtual concerts. The concept of "social VR" shifts from niche tech demo to genuine community infrastructure. Full Body Tracking gains mainstream community awareness.
2021 — Maturity
UDON 2.0, Creator Economy Foundations
VRChat releases significant UDON improvements. World interactivity becomes dramatically more sophisticated — full games, physics engines, synchronized multiplayer logic, and creative tools emerge. The community starts to professionalize. Top avatar creators are earning real income. VRChat announces a $100M+ funding milestone (combined rounds).
2022 — The EAC Controversy
Easy Anti-Cheat Ends the Mod Era
VRChat implements Easy Anti-Cheat (EAC), effectively ending the era of third-party client modifications. The popular VRCMods ecosystem — which added features like avatar search, mirror controls, and desktop QoL improvements — is killed overnight. Community backlash is significant but VRChat holds the decision. The VRChat Creator Companion (VCC) launches as the official approved modding alternative.
2023 — Safety & Adult Content
Content Gating, Layered Safety Rollout
VRChat introduces content gating for adult worlds, new Trust & Safety team expansions, and improved reporting infrastructure. The platform begins confronting the dual reality of its massive user base: families and vulnerable users sharing infrastructure with adult content and bad actors. New Safety Bubble defaults protect users by default.
2024 — Mobile Era
VRChat Goes Mobile
VRChat officially launches on iOS and Android, dramatically expanding the potential user base beyond headset owners. Mobile users can attend events, hang out, and communicate without any VR hardware. The Avatar Marketplace development begins internally. Platform reaches an estimated 40M+ registered accounts.
2025 — Creator Economy Formalized
Avatar Marketplace Launch + Age Verification
May 2025: VRChat launches its native in-app Avatar Marketplace, allowing creators to sell directly to users without leaving the platform. January 2025: Platform-level age verification rolls out for 18+ content access. The creator economy matures — top creators are earning five figures monthly. $95.2M total funding across all rounds publicly confirmed.
2026 — A New Record
156,700 Concurrent Users — NYE Sanrio Concert
New Year's Eve 2026. A Sanrio-themed anime concert becomes the largest virtual event in VRChat history — 156,700 concurrent users in a single platform event. The number shatters every previous record and positions VRChat as one of the largest live music venues on Earth, virtual or physical. The future is undeniable.
