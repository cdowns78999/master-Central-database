Wings Dashboard · Firebase Setup

Push Notification Setup — 7 Steps

Complete these steps in order. Check each one off as you go.

Progress 0 / 7 complete
1
Create the Firebase Project
Go to console.firebase.google.com → click "Add project" → name it Wings Dashboard → disable Google Analytics → hit Create.
2
Register the Web App & Copy firebaseConfig
Inside the project → click "Add app" (web </> icon) → register app name "Wing Dashboard" → copy the full firebaseConfig object shown on screen.
// firebaseConfig example — yours will have real values const firebaseConfig = { apiKey: "AIzaSy...", authDomain: "wings-dashboard.firebaseapp.com", projectId: "wings-dashboard", storageBucket: "wings-dashboard.appspot.com", messagingSenderId: "1234567890", appId: "1:123:web:abc..." };
3
Generate the VAPID Public Key
Project Settings (⚙️ gear icon) → Cloud Messaging tab → Web Push Certificates section → click "Generate key pair" → copy the VAPID public key string.
// VAPID key looks like this (save it — you'll need it in step 5) vapidKey: "BLc3J8X7...long base64 string..."
4
Download the Service Account JSON
Project Settings → Service Accounts tab → click "Generate new private key" → save the downloaded JSON to your Wing Dashboard folder at the path below.
ADMIN/tools/firebase-service-account.json ↑ drop the downloaded file here (rename it if needed)
5
Populate firebase-config.json
Open ADMIN/tools/firebase-config.json → paste in all values from steps 2–4.
{ "projectId": "wings-dashboard", "apiKey": "AIzaSy...", "appId": "1:123:web:abc...", "messagingSenderId": "1234567890", "vapidKey": "BLc3J8X7...", "serviceAccountPath": "ADMIN/tools/firebase-service-account.json" }
6
Replace Placeholders in Homebase Files
Open both files below and swap every YOUR_* placeholder with the real firebaseConfig values from step 2.
ADMIN/homebases/thennow/index.html ADMIN/homebases/thennow/firebase-messaging-sw.js // Search for and replace all of these: YOUR_API_KEY → real apiKey YOUR_AUTH_DOMAIN → real authDomain YOUR_PROJECT_ID → real projectId YOUR_MESSAGING_SENDER_ID → real messagingSenderId YOUR_APP_ID → real appId YOUR_VAPID_KEY → real vapidKey
7
Redeploy Homebase & Start Push Server
Deploy the updated homebase to Netlify, then start the push server — two commands in terminal.
# 1. Deploy homebase to Netlify (production) netlify deploy --prod --dir=ADMIN/homebases/thennow # 2. Start the push notification server python ADMIN/tools/push-server.py
🔔

All 7 steps done? You're live.

Head to the Wing Dashboard → find any client card → click the 🔔 bell icon on it. A real push notification will fire to that client's device — Firebase handles the delivery end-to-end.
