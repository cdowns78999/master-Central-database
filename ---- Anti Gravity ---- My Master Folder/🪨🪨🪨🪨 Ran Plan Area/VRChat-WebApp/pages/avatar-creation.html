Avatar Creation Pipeline

1

Concept & Design

Sketch your character. Decide on style: anime, realistic, furry, mech, or original. Reference boards help.

2

Model

Build the 3D model in VRoid Studio (beginner-friendly) or Blender (full control). BOOTH bases can shortcut this step.

3

Rig & Export

Use CATS Blender Plugin to fix bones, optimize mesh, and export as FBX. Keep polygon count under limits.

4

Unity Import

Open in Unity 2022.3.22f1 (required version). Import FBX, set up materials, configure humanoid rig.

5

VCC + SDK Setup

Install VRChat Creator Companion (VCC). Add Avatars SDK to your Unity project. Set up descriptor and viewpoint.

6

Upload to VRChat

Build & publish from Unity. Test in-game. Configure fallback avatar for Quest users if PC-only.

Tools by Skill Level

VRoid Studio
Free
Drag-and-drop character maker by Pixiv. Anime-style avatars with full customization. Export as VRM, convert to FBX for VRChat.
Blender
Free
Industry-standard 3D modeling. Full mesh control, weight painting, UV mapping. Steep learning curve but unlimited power.
CATS Blender Plugin
Free
One-click avatar optimization for VRChat. Fixes armature, merges meshes, decimates polygons. Essential for most workflows.
Modular Avatar
Free
Non-destructive avatar assembly in Unity. Swap outfits, accessories, and components without breaking your base avatar.
VRCFury
Free
Advanced avatar toolbox. Gesture toggles, contact receivers, physbone automation. Drag-and-drop component system.
Substance Painter
Paid ($$$)
Professional texturing tool by Adobe. PBR materials, smart masks, projection painting. Industry standard for high-quality skins.

🛍️ BOOTH Ecosystem

Avatar bases on BOOTH (booth.pm) typically cost ¥3,000–¥8,000 ($20–$55 USD). These include fully rigged, upload-ready models with multiple outfit options. Japanese creators dominate the marketplace. Most bases are Blender-compatible with Unity packages included.

📱 Quest Compatibility

Quest avatars must stay under 10,000 polygons, 2 material slots, and no custom shaders. Use the VRChat Performance Ranking system in Unity to check. Many PC avatars need a separate Quest-optimized version — or set a fallback avatar so Quest users see a default model instead of a gray robot.
