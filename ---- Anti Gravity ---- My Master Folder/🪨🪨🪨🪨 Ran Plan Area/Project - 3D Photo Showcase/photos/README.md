# Photos

Drop photos here as `01.jpg`, `02.jpg`, ... `12.jpg`.

## Guidelines
- **Count:** 12 photos total
- **Naming:** zero-padded, two-digit (`01.jpg` through `12.jpg`)
- **Aspect ratio:** 16:9 recommended
- **Format:** `.jpg` preferred (`.png` works but heavier)
- **Resolution:** 1920x1080 or larger — will scale down on smaller screens
- **File size:** aim for under 500 KB per image for smooth loading

## Manifest
The list of expected filenames lives in `manifest.json`. Update it if you change the count or filenames.
