# Big Copy - Chrome Extension

A simple Chrome extension that copies the entire page content with one click.

## Installation

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" (toggle in the top right)
3. Click "Load unpacked"
4. Select the `big copy` folder
5. The extension is now installed!

## Usage

1. Navigate to any webpage
2. Click the Big Copy extension icon in your toolbar
3. Choose what to copy:
   - **Include HTML**: Copies the full HTML source code
   - **Include Text**: Copies all visible text content
4. Click "Copy Entire Page"
5. The content is now in your clipboard!

## Features

- Copy full HTML source code
- Copy all text content
- Choose to copy HTML, text, or both
- Simple, clean interface
- Fast and lightweight
