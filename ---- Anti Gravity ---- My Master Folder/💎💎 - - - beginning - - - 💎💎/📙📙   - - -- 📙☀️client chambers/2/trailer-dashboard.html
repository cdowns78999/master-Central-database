How to make this
Follow in order. Steps 3 + 4 run at the same time.
1
Starfield + Logo
Prompt Veo for a 5s starfield with logo fade-in. Record a 3s intro in Nano. Drop both into Pika, sync, freeze last frame, export.
Veo Nano Pika ~8 min
2
Build the HTML
Claude Code builds the screenshare page and outro card in one session. Copy a backup. Both pages ready to go.
Claude Code ~12 min
3
Screen Record
Open the page in Chrome, hit Loom with mic on, walk through it. Drop into Pika, trim dead air, add transition, export.
Loom Pika parallel ~8 min
4
Scene 3 Content
Decide what this scene is. Prompt Veo for visuals while you record VO in Nano. Runs alongside Step 3.
Veo Nano parallel ~5 min
5
Outro Card
GoFullPage screenshots the outro HTML. Pika adds a 2s fade. Quick — it's just a card.
GoFullPage Pika ~3 min
6
Stitch + Export
Drop all 4 clips into Pika. Timeline order, add transitions, export MP4 at 1080p. Done.
Pika ~10 min
