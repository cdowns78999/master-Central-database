⚡
AAO
What: Avatar Optimizer — non-destructive optimization utilities

Why: Part of the NDMF ecosystem — zero-destructive, one-click optimization

d4rk Optimizer
What: Reduces skinned mesh & material count before upload
How: Add component → "Apply on Upload" — non-destructive, creates optimized copy
Why: Simple, reliable optimizer — safe because original is never modified

LazyOptimiser
What: Auto-optimizes on avatar build — strips unused geometry
How: Removes unused GOs, blendshapes, merges meshes, combines materials
Why: "Be lazy but optimized" — set it and forget it pipeline
🎃
Pumkin Tools
What: Unity editor tool — avatar stats, viseme filling, eye tracking setup
How: Import into Unity, access from toolbar — works alongside SDK
Why: VRChat docs officially recommends it for common avatar setup tasks

How: Install as Blender addon — supports MMD, Mixamo, DAZ, and more formats
Why: The most widely used Blender tool for VRChat — 7-hour pipeline → minutes

Shadster Tools
What: Unity editor tools — physbone setup, Gogo Loco integration, shapekey gen
How: Import package, toolbar menu → bulk operations on avatar hierarchy
Why: Speeds up repetitive Unity avatar setup tasks significantly

Gogo Loco
What: Advanced locomotion system — sit, lay, crawl, dance in VRChat
How: Unity animator layer + gesture menu — works with any humanoid avatar
Why: The most popular locomotion add-on — transforms how you move in-world

I5UnityTools
What: Opinionated Unity tools — texture optimization, multi-uploader

Modular Avatar
What: Drag-and-drop avatar assembly on the NDMF framework

Why: The #1 NDMF tool — non-destructive, modular, composable avatar building

Poiyomi Shader
What: Most popular toon shader in VRChat — massive feature set
How: Install via VCC → assign to avatar materials, tweak in inspector
Why: The go-to shader for stylized avatars — SSS, decals, custom lighting
