Step 5 — Upload Checklist

Before you build and publish — verify everything is ready.

  • Unity version matches VCC recommendation
  • Avatar Descriptor component added (view position, visemes, eye look)
  • Shaders installed and assigned to all materials
  • PhysBones configured for hair, clothing, accessories
  • Expression menu and parameters set up
  • Optimization tools applied (AAO / d4rk / LazyOptimiser)
  • Quest fallback version created (if cross-platform)
  • Blueprint ID copied for Quest version
  • No errors in SDK Build Control Panel
  • Build & Publish for Windows → then Quest
