# ChatGPT Copy - Chrome Extension

A simple, one-click extension that copies a specific refinement prompt for ChatGPT.

## Purpose
Copies the following text to your clipboard:
> "Adopt a more practical, sensible tone, while maintaining a deep, warm, almost pseudo-spiritual 'i know im on the right path' essence of thoughtfulness of the article"

## Usage
1. Click the icon (📋) in your toolbar.
2. The script is automatically copied.
3. The popup closes itself instantly.
4. Paste into ChatGPT.

## Installation
1. Open `chrome://extensions/`.
2. Enable **Developer mode**.
3. Click **Load unpacked**.
4. Select this folder (`ChatGPT Copy`).
